# Emotion Restaurant Recommend System
유아 영어학습 플랫폼

## 소개
유아 대상 영어 콘텐츠가 부족하고, 기존 서비스는 단순 강의 중심이라 아이들의 집중 유지가 어려웠습니다. 
이를 보완하기 위해 강사가 다양한 자료를 업로드하고, 유아의 수준에 맞춘 학습을 제공할 수 있는 플랫폼을 기획했습니다.
